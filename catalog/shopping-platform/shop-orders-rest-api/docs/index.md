# shop-orders-rest-api

REST API exposed by `shop-orders-service` for order management.

- **Type:** openapi
- **Provider:** `shop-orders-service`

## Endpoints

| Method | Path           | Description       |
| ------ | -------------- | ----------------- |
| GET    | `/orders`      | List orders       |
| POST   | `/orders`      | Create a new order |
| GET    | `/orders/{id}` | Get order detail  |

The full OpenAPI definition is embedded in the API entity and viewable in the Backstage **Definition** tab.
