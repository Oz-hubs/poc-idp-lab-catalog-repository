# shop-web

Web frontend for the `shopping-platform` system.

- **Type:** website
- **Stack:** React (example)
- **Consumes:** `shop-orders-rest-api`

## Responsibilities

- Renders the product catalog and checkout UI.
- Calls the orders REST API to create and track orders.

## Local development

This is an example entity — no real source yet. To wire a real repository, create `Oz-hubs/shop-web` on GitHub.
